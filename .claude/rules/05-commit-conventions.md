# Rule: コミット規約

## Priority: MEDIUM

## コミットメッセージ

- コミットメッセージは日本語で記述する
- 以下のプレフィックスを使用する:
  - `spec:` - 仕様ドキュメントの追加・変更
  - `test:` - テストケースの追加・変更
  - `impl:` - コンパイラ実装の追加・変更
  - `fix:` - バグ修正
  - `refactor:` - リファクタリング
  - `chore:` - その他 (ビルド設定、CI等)

## 例

```
spec: 型システムの仕様を追加
test: 整数型の基本演算テストケースを追加
impl: 字句解析器を実装
fix: 文字列リテラルのエスケープ処理を修正
```
