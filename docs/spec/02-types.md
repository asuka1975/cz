# 02 - 型システム

## 概要

Cz は静的型付け言語である。MS1 では `i32` (32 ビット符号付き整数) のみを型として提供する。

## 構文

```bnf
type = "i32"
```

## 意味論

### i32

- 32 ビット符号付き整数型
- 値の範囲: -2,147,483,648 から 2,147,483,647
- 整数リテラル、算術演算、比較演算の結果として生成される

### 真偽値の扱い

MS1 では独立した `bool` 型は存在しない。条件式 (`if` の条件、`while` の条件) では以下のいずれかを使用する:

- 比較演算子の結果 (`==`, `!=`, `<`, `>`, `<=`, `>=`)
- 論理演算子の結果 (`&&`, `||`, `!`)
- リテラル `true`, `false`

内部的には `true` は `1`、`false` は `0` として `i32` で表現される。

### 型の一致

すべての変数宣言、関数の戻り値、演算において型の一致が要求される。MS1 では `i32` しか存在しないため、型の不一致は将来のマイルストーンで意味を持つ。

## 例

```cz
let x: i32 = 42;
let y: i32 = x + 1;
let flag: i32 = true;  // true は 1 として扱われる
```

## 制約・制限

- MS1 では `i32` のみ。他の整数型 (`i8`, `i16`, `i64`, `u32` 等) は将来対応
- 浮動小数点型は MS1 ではサポートしない
- 整数オーバーフローの動作は未定義 (将来のマイルストーンで定義予定)
- 将来的に `bool` 型を独立した型として導入する可能性がある
